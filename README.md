# SMART-TRAFFIC-INTELLIGENCE-SYSTEM
# 🚦 Smart Traffic Congestion Intelligence System

## 📌 Overview
The **Smart Traffic Congestion Intelligence System** is a Machine Learning-based project designed to analyze traffic data and predict congestion levels (Low, Medium, High).  
This system helps in understanding traffic patterns and supports better traffic management decisions.

---

## 🎯 Objectives
- Predict traffic congestion levels using ML models
- Analyze traffic patterns based on time and environmental conditions
- Provide an interactive dashboard for real-time predictions
- Visualize traffic trends using graphs

---

## 🧠 Machine Learning Concepts Used

- Supervised Learning
- Unsupervised Learning
- Feature Engineering
- Cyclic Feature Transformation
- K-Means Clustering
- Random Forest Classification
- Linear Regression
- Model Evaluation (Accuracy, MSE)

---

## ⚙️ Technologies Used

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit (UI Dashboard)

---

## 📊 Dataset

We used the **Metro Interstate Traffic Volume Dataset**, which includes:
- Traffic volume
- Temperature
- Weather conditions
- Date and time

---

## 🔄 Project Workflow

1. Data Collection  
2. Data Preprocessing  
3. Feature Engineering  
4. Clustering (K-Means)  
5. Model Training (Random Forest)  
6. Prediction  
7. Visualization  
8. Deployment using Streamlit  

---

## 💻 Features

- 🚗 Traffic Prediction (Low / Medium / High)
- 🎛️ Interactive Dashboard (Streamlit UI)
- 📊 Live Graph Visualization
- 🌙 Dark Mode UI
- 🔍 Confidence Score Display
